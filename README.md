# StudyFlow 📚

StudyFlow is a simple productivity web app that I built to help students manage their daily study tasks more effectively.  
Instead of using multiple apps for tasks, notes, and timers, this project combines everything into one clean interface.

The goal of this project was not just to build something useful, but also to improve my understanding of JavaScript and maintain consistent GitHub contributions.


## What this app can do

### Task Management
You can add your study tasks, delete them, and mark them as completed.  
All tasks are saved in the browser, so even if you refresh the page, your data will still be there.

### Pomodoro Timer
The app includes a simple 25-minute study timer.  
You can start, pause, and reset it depending on your study session.

### Notes Section
There’s a small notes area where you can write anything important while studying.  
Your notes are also saved automatically in local storage.

### Progress Overview
The app shows:
- total number of tasks  
- number of completed tasks  

This helps you track how productive you are.

### Dark Mode
You can switch between light and dark mode.  
The app remembers your preference.

### Backup (Export / Import)
You can download your data as a file and upload it later.  
This is useful if you switch devices or want to keep a backup.


## Technologies used

- HTML
- CSS
- JavaScript (Vanilla JS)
- LocalStorage


## How to run the project

1. Download or clone the repository
2. Open the folder
3. Double-click on `index.html`

That’s it. No setup needed.
